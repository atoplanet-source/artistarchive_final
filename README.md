# Artist Archives

A curated visual dataset of rare, high-quality imagery spanning fashion, beauty, travel, lifestyle, celebrity, product, and portraiture — sourced directly from the personal archives of elite commercial photographers.

## About

Artist Archives is the source for one of the world's most comprehensive datasets of professional photography and video for AI training and machine learning.

- **Exclusive Access** — Never published, never scraped — sourced directly from creators
- **Metadata-Rich & Organized** — Hybrid AI + human labeling, quality-controlled assets
- **Diverse & Authentic** — Global content from across decades, regions, and industries
- **Ethically Sourced** — Rights-cleared, legally licensable, and ready for compliant training
- **Production Ready** — Curated subsets or full-archive access delivered securely at enterprise scale

## Tech Stack

- HTML5 / CSS3
- Tailwind CSS (CDN)
- Vanilla JavaScript

## Development

This is a static site. To run locally:

```bash
# Using Python
python -m http.server 8000

# Or using Node
npx serve
```

## Deployment

Deployed via Netlify. Connected to GitHub for automatic deployments on push to `main`.
